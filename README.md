# EnchantTransfer
EnchantTransfer Plugin

EnchantTransfer is a Minecraft plugin that allows players to transfer enchantments between items for a monetary cost. The plugin supports transferring enchantments across tools, swords, armor, and even paper for creating enchantment books. Itâ€™s perfect for servers looking to add a unique way to manage enchantments while encouraging in-game economy usage.

Features

Transfer enchantments between compatible items (e.g., sword to sword, armor to armor).

Convert enchantments from items into enchantment books (via paper).

Charge players in-game currency for each transfer.

Fully configurable costs for transfers and supported items.

Supports major economy plugins via Vault integration.

Configurable messages for customization.
